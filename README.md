# Real-Time-AI-Workout-Assistant
A real-time AI vision system that automatically detects your exercise type and counts your repetitions to provide hands-free workout monitoring.
This Real-Time AI Workout Assistant is an end-to-end computer vision application that transforms a standard webcam into an intelligent digital personal trainer. Built entirely in Python, the pipeline leverages MediaPipe BlazePose for 3D human landmark extraction and a custom PyTorch Multi-Layer Perceptron (MLP) to accurately classify exercise states in real-time. To guarantee robust performance regardless of camera distance or user positioning, the system features a mathematical preprocessing layer that normalizes raw skeletal coordinates to achieve complete translation and scale invariance. Finally, a programmatic Finite State Machine (FSM) filters the neural network's predictions to track dynamic movement phases (concentric/eccentric) and reliably count completed repetitions, projecting the live metrics via an OpenCV Heads-Up Display (HUD).
